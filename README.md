# Java POO – Conta Bancária

## 📌 Descrição
Projeto em Java desenvolvido para praticar Programação Orientada a Objetos (POO),
simulando uma conta bancária com menu interativo no console.

## 🚀 Funcionalidades
- Criar conta bancária
- Depositar valores
- Sacar valores
- Exibir dados da conta
- Menu interativo

## 🛠️ Tecnologias
- Java
- Programação Orientada a Objetos
